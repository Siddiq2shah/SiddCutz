Overview

SiddCutz is a full-stack web application developed to manage and analyze appointment bookings for a personal barber business.
It combines AI-driven scheduling, data analytics, and operational automation to improve client management and service efficiency.

Features

Developed an integrated appointment scheduling system using PHP, MySQL, and HTML/CSS for seamless client booking and tracking.

Designed an AI-based recommendation system that analyzes booking data to optimize time slots and predict high-demand hours.

Built Python scripts to clean, process, and visualize customer booking data for decision-making and operational improvement.

Analytics & Visualization

Applied data analytics techniques to track booking frequency, identify peak service times, and visualize customer trends.

Leveraged Google Data Analytics principles (data cleaning, aggregation, visualization) to generate insights from historical booking records.

Used dashboards to monitor daily appointments, service types, and client retention, supporting data-driven business optimization.

Tech Stack

Frontend: HTML, CSS, Bootstrap

Backend: PHP, MySQL

AI / Analytics: Python, Pandas, TensorFlow

Visualization: Matplotlib, Google Data Studio

Impact

Reduced scheduling conflicts and idle time through data-backed recommendations.

Improved client experience and booking consistency by 20% through operational optimization.

Enabled better decision-making via real-time visualization of key metrics.
